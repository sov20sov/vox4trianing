# Vox Training Center

A modern, responsive React application for Vox Training Center, providing information about courses, instructors, and services. Built with React, TypeScript, and Vite for optimal performance and developer experience.

## Features

- **Home Page**: Welcome section with overview of the training center
- **Courses**: Detailed information about available courses including IELTS preparation
- **Instructors**: Profiles and information about qualified instructors
- **About**: Information about the training center's mission and values
- **Contact**: Contact form and information for inquiries
- **Multilingual Support**: Built-in translation support for multiple languages
- **Responsive Design**: Optimized for desktop and mobile devices
- **Smooth Animations**: Enhanced user experience with Framer Motion

## Technologies Used

- **React 19**: Modern React with hooks and concurrent features
- **TypeScript**: Type-safe JavaScript for better development experience
- **Vite**: Fast build tool and development server
- **React Router**: Client-side routing for single-page application
- **Framer Motion**: Animation library for smooth transitions
- **Lucide React**: Modern icon library
- **CSS**: Custom styling with responsive design

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd vox-for-training-center
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
